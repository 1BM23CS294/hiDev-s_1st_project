# **App Name**: CareerMatch AI

## Core Features:

- Resume Upload & Management: Allow users to upload resume files (e.g., PDF, DOCX) for processing and view their uploaded resumes.
- Job Description Input: Provide an intuitive interface for users to input or paste job description text.
- AI-Powered Resume Parsing: An AI tool that accurately extracts and structures key information (e.g., skills, experience, education, contact) from uploaded resumes.
- AI-Driven Skills Matching & Scoring: An AI tool that intelligently compares extracted resume data with job requirements to calculate a comprehensive match score (0-100).
- AI-Generated Hiring Recommendations: An AI tool that provides specific, data-driven hiring recommendations and insights based on the match score and parsed candidate data.
- Detailed Analysis Reports: Display a clear, professional, and user-friendly report presenting extracted data, match scores, and recommendations.
- Candidate Profile and Report Storage: Securely store processed resume data and generated analysis reports in a structured (JSON) format for later retrieval and review.

## Style Guidelines:

- Primary color: A modern and trustworthy deep blue (#2669E0) to convey professionalism and technological efficiency.
- Background color: A very subtle, almost white, cool-toned blue (#F6F8FB) to ensure readability and a clean interface.
- Accent color: A vibrant, clear cyan (#33CCFF) to highlight interactive elements and important information, creating visual dynamism without being overwhelming.
- Headline and body font: 'Inter' (sans-serif) for its modern, objective, and highly legible appearance, suitable for displaying structured data and reports.
- Utilize clean, crisp line icons that are intuitive and maintain a professional, data-focused aesthetic.
- Employ a structured and spacious layout with clear visual hierarchy, optimizing for readability of data, forms, and analytical reports across different devices.
- Implement subtle and purposeful animations for transitions, loading states, and user feedback, enhancing perceived responsiveness and user experience.
